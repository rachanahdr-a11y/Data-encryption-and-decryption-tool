# Data Encryption and Decryption Tool

This is a web-based tool that allows users to encrypt and decrypt text data using AES encryption provided by the CryptoJS library. The tool features a modern and user-friendly interface with light and dark mode support, enhancing usability and visual appeal.

## Features

- **AES Encryption and Decryption:** Securely encrypt and decrypt text using a user-provided key.
- **Dark Mode and Light Mode:** Toggle between dark and light themes for comfortable viewing in different lighting conditions.
- **Show/Hide Encryption Key:** Easily toggle the visibility of the encryption key input for convenience and security.
- **Glassmorphic UI Design:** Modern glass-like container with smooth gradients and shadows for an elegant look.
- **Responsive and Interactive:** Smooth transitions, hover animations, and clear error handling for a seamless user experience.

## How It Works

1. **Encryption:**
   - Enter an encryption key in the "Encryption Key" input field.
   - Enter the text you want to encrypt in the "Input Text" area.
   - Click the "Encrypt" button.
   - The encrypted text will appear in the "Output" area.

2. **Decryption:**
   - Enter the decryption key (should be the same as the encryption key) in the "Encryption Key" input field.
   - Enter the encrypted text in the "Input Text" area.
   - Click the "Decrypt" button.
   - The original decrypted text will appear in the "Output" area.

3. **Dark Mode:**
   - Use the toggle switch at the top right to switch between light and dark modes.
   - **User experience is better when using dark mode** due to reduced eye strain and enhanced contrast.

4. **Show/Hide Key:**
   - Click the "Show" button next to the encryption key input to toggle visibility of the key.

## User Experience

- The tool is designed to be intuitive and visually appealing.
- The dark mode provides a better user experience by reducing eye strain and improving readability.
- Input fields and buttons have clear visual feedback and animations.
- Error messages guide users to provide valid inputs and keys.

## Technologies Used

- HTML, CSS, JavaScript
- CryptoJS library for AES encryption/decryption
- Google Fonts for custom typography

## How to Use

- Open the `index.html` file in a modern web browser.
- Enter your encryption key and text.
- Use the buttons to encrypt or decrypt.
- Toggle dark mode for a better viewing experience.
